# flexcast - Application for sampling, merging and manipulating music directly in your browser.

## Requirements
Only Node.js

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm start
```

It will be locally hosted in `http://127.0.0.1:8080`

### Build for production
```
npm run build
```
